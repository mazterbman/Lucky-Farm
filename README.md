Проект: 3D-прототип фермы (а-ля Fancy Farm)
Статус: прототип, активная разработка

📝 Описание
Этот проект представляет собой прототип игры в жанре «ферма», вдохновлённый Fancy Farm, но с полным 3D-пространством. Игрок может взаимодействовать с объектами, собирать ресурсы, выращивать культуры и развивать ферму.

Проект создан для отработки архитектуры, асинхронной загрузки и системы зависимостей в Unity.

🛠️ Используемые технологии
ZenJect – внедрение зависимостей (DI) для модульности и тестируемости кода

UniTask – асинхронное программирование без корутин (легковесные задачи)

Addressables – асинхронная загрузка и управление контентом (префабы, сцены, ассеты)

🎮 Особенности прототипа
Полностью 3D-окружение, свободная камера

Система посадки и сбора урожая

Инвентарь и взаимодействие с объектами

DI-контейнер для сервисов (менеджер фермы, экономики, загрузки)

Асинхронная загрузка уровней/объектов через Addressables

Использование UniTask для плавных анимаций и задержек без GC-аллокаций

🖥️ Как собрать и запустить
Убедитесь, что установлен Unity 2021.3 LTS или новее.

Склонируйте репозиторий:

bash
git clone https://github.com/ваш-аккаунт/название-проекта.git
Откройте проект в Unity.

Установите через Package Manager зависимости (если не подтянулись автоматически):

com.cysharp.unitask

com.unity.addressables

Дождитесь импорта ZenJect (если используется .unitypackage – установите вручную).

Откройте сцену Assets/Scenes/Game.unity.

Нажмите Play.
